# Bag Map Visualizer

这个工具提供了一个功能丰富的交互式界面，使用Open3D可视化从ROS bag文件中读取的地图数据。

## 主要功能

### 基础功能
- ✅ 从ROS bag文件读取`/keyframe_pcd`和`/keyframe_pose`话题
- ✅ 将点云按照对应的位姿变换到全局坐标系
- ✅ 支持体素下采样以减少内存使用
- ✅ 交互式3D可视化界面

### 可视化模式
- 🎨 **彩色关键帧模式**：每个关键帧使用不同的颜色显示
- 📊 **合并点云模式**：所有关键帧合并为单一点云
- 🌈 **高度着色**：根据Z坐标进行颜色映射（蓝色→红色）
- 🛤️ **轨迹显示**：显示关键帧位姿轨迹线和位置标记

### 编辑功能
- ✂️ **裁剪模式**：选择要保留的区域
- 🗑️ **删除模式**：选择要删除的区域
- 💾 **保存功能**：将编辑后的点云保存为PCD文件

## 安装依赖

```bash
pip install open3d numpy rosbag
```

或使用requirements文件：

```bash
pip install -r requirements.txt
```

## 使用方法

### 基本用法

```bash
python3 bag_map_visualizer.py /path/to/your/map.bag
```

### 指定体素大小

```bash
# 使用0.5米体素下采样（更精细）
python3 bag_map_visualizer.py /path/to/your/map.bag --voxel_size 0.5

# 使用2.0米体素下采样（更快速）
python3 bag_map_visualizer.py /path/to/your/map.bag --voxel_size 2.0

# 禁用下采样（显示所有点，可能很慢）
python3 bag_map_visualizer.py /path/to/your/map.bag --voxel_size 0
```

## 交互式控制

### 键盘快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **K** | 切换彩色关键帧模式 | 在"彩色关键帧"和"合并点云"视图之间切换 |
| **T** | 切换轨迹显示 | 显示/隐藏关键帧轨迹线和位置标记 |
| **H** | 切换高度着色 | 在"高度着色"和"统一颜色"之间切换 |
| **C** | 裁剪模式 | 框选要保留的区域 |
| **X** | 删除模式 | 框选要删除的区域 |
| **S** | 保存点云 | 将当前点云保存为`edited_map.pcd` |
| **R** | 重置视角 | 恢复默认相机视角 |
| **F** | 全屏切换 | 切换全屏模式 |
| **P** | 打印相机参数 | 在终端显示当前相机参数 |
| **?** | 显示帮助 | 在终端显示所有快捷键 |
| **Q/ESC** | 退出 | 关闭可视化窗口 |

### 鼠标控制

| 操作 | 功能 |
|------|------|
| 左键拖动 | 旋转视角 |
| 右键拖动 | 缩放视图 |
| 中键拖动 | 平移视图 |
| 滚轮 | 放大/缩小 |

## 详细功能说明

### 1. 彩色关键帧模式 (K键)

**用途**：查看各个关键帧的分布和重叠情况

- 每个关键帧使用独特的颜色显示
- 便于识别不同时间采集的数据
- 可以观察到关键帧之间的重叠区域

**使用场景**：
- 检查建图质量
- 识别重复扫描区域
- 分析关键帧分布

### 2. 轨迹显示 (T键)

**特性**：
- 黄色线段连接相邻关键帧
- 红色球体标记关键帧位置
- 清晰显示机器人运动路径

**使用场景**：
- 分析机器人运动轨迹
- 检查轨迹连续性
- 识别回环位置

### 3. 高度着色 (H键)

**颜色映射**：
- 蓝色：低海拔
- 绿色：中等海拔
- 红色：高海拔

**使用场景**：
- 地形分析
- 高程可视化
- 多层结构识别

### 4. 点云编辑

### 适用场景
- ✅ 删除大片区域（如地图边缘噪点）
- ✅ 按空间范围快速裁剪
- ✅ 批量处理规则区域

### 操作步骤

#### 1️⃣ 启动可视化工具
```bash
python3 bag_map_visualizer.py your_map.bag --voxel_size 1.0
```

#### 2️⃣ 查看点云范围
观察点云，程序会显示当前坐标范围：
```
当前点云范围:
  X: [-50.23, 123.45]
  Y: [-30.12, 89.67]
  Z: [-2.34, 15.78]
```

#### 3️⃣ 删除点云（按X键）

**选项1: 球形删除**
```
选择删除模式 [1/2/3]: 1

输入要删除的球形区域:
  中心X坐标 [0.00]: 50.0      ← 输入要删除区域的中心点
  中心Y坐标 [0.00]: 30.0
  中心Z坐标 [0.00]: 0.0
  半径: 10.0                    ← 输入半径（米）

✓ 删除完成! 删除了 15230/500000 个点
```

**适用场景**：
- 删除某个物体周围的点（如车辆、行人）
- 清理圆形区域的噪点
- 移除特定位置的聚集点

**选项2: 方形删除**
```
选择删除模式 [1/2/3]: 2

输入要删除的方形区域:
  X最小值: -10.0              ← 定义立方体范围
  X最大值: 10.0
  Y最小值: -5.0
  Y最大值: 5.0
  Z最小值: -1.0
  Z最大值: 3.0

✓ 删除完成! 删除了 23456/500000 个点
```

**适用场景**：
- 删除地图边缘区域
- 移除特定高度范围的点（如天花板）
- 清理矩形区域

#### 4️⃣ 裁剪点云（按C键）

```
输入要保留的范围 (按Enter使用当前最大范围):
  X最小值 [-50.23]: -40.0    ← 定义保留区域
  X最大值 [123.45]: 100.0
  Y最小值 [-30.12]: -20.0
  Y最大值 [89.67]: 80.0
  Z最小值 [-2.34]: -2.0
  Z最大值 [15.78]: 10.0

✓ 裁剪完成! 保留了 450000/500000 个点
```

**适用场景**：
- 提取感兴趣区域（ROI）
- 移除地图外围所有点
- 按高度裁剪（如只保留地面）

#### 5️⃣ 保存结果（按S键）
```
✓ 点云保存到 edited_map.pcd
```

## 可视化示例

### 示例1：查看地图整体结构

```bash
python3 bag_map_visualizer.py kitti.bag --voxel_size 1.0
```

启动后：
1. 默认显示合并点云（高度着色）+ 轨迹
2. 按**K**切换到彩色关键帧模式查看详细结构
3. 按**T**隐藏轨迹以获得更清晰的点云视图

### 示例2：编辑和清理地图

```bash
python3 bag_map_visualizer.py kitti.bag --voxel_size 0.5
```

编辑流程：
1. 按**X**进入删除模式
2. 框选要删除的噪点或无用区域
3. 按**X**确认删除
4. 重复步骤1-3直到清理完成
5. 按**S**保存编辑后的地图

### 示例3：精细分析特定区域

```bash
python3 bag_map_visualizer.py kitti.bag --voxel_size 0.3
```

分析流程：
1. 使用鼠标旋转到感兴趣的视角
2. 按**K**切换到彩色关键帧模式
3. 按**H**切换颜色模式以获得更好的对比度
4. 按**P**打印当前相机参数（可用于后续重现视角）

## 输出信息

程序会在终端显示：

- 关键帧数量和处理进度
- 下采样前后的点数统计
- 点云空间范围（X/Y/Z最小值和最大值）
- 操作反馈（切换模式、保存文件等）
- 相机参数（按P键时）

## 性能建议

### 大型地图（>1000万点）
```bash
python3 bag_map_visualizer.py large_map.bag --voxel_size 2.0
```
- 使用较大的体素尺寸
- 首先在合并模式下浏览
- 需要时才切换到彩色关键帧模式

### 中型地图（100万-1000万点）
```bash
python3 bag_map_visualizer.py medium_map.bag --voxel_size 1.0
```
- 默认体素尺寸即可
- 所有功能流畅运行

### 小型地图（<100万点）
```bash
python3 bag_map_visualizer.py small_map.bag --voxel_size 0.3
```
- 可以使用较小的体素尺寸
- 获得更精细的显示效果

## 故障排除

### 问题：窗口显示空白
- **原因**：可能没有有效的点云数据
- **解决**：检查bag文件中是否包含正确的话题和数据

### 问题：切换彩色关键帧模式很慢
- **原因**：关键帧数量太多或点数太多
- **解决**：使用更大的`--voxel_size`参数

### 问题：编辑操作不响应
- **原因**：可能在彩色关键帧模式下
- **解决**：编辑操作会自动切换到合并模式，再次尝试

### 问题：保存的文件很大
- **原因**：点云未经充分下采样
- **解决**：使用更大的体素尺寸重新运行

### 问题：找不到bag文件
- **原因**：路径错误或文件不存在
- **解决**：
  ```bash
  # 使用绝对路径
  python3 bag_map_visualizer.py /home/user/maps/kitti.bag
  
  # 或先进入目录
  cd /home/user/maps
  python3 bag_map_visualizer.py kitti.bag
  ```

## 技术细节

### Bag文件要求
- 必须包含`/keyframe_pcd`话题（`sensor_msgs/PointCloud2`类型）
- 必须包含`/keyframe_pose`话题（`geometry_msgs/PoseStamped`类型）
- 两个话题的消息数量必须匹配

### 坐标系
- 所有点云会变换到全局地图坐标系
- 轨迹显示关键帧的全局位置
- 坐标轴显示：X(红)、Y(绿)、Z(蓝)

### 内存管理
- 体素下采样可以显著减少内存使用
- 彩色关键帧模式会占用更多内存
- 编辑操作会创建点云副本作为备份
